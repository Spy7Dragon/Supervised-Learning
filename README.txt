# Supervised Learning
 
Download repository from https://github.com/Spy7Dragon/Supervised-Learning.

Unzip the download. From root folder of download run 
'py -2 main.py' from a terminal window.

Data is from https://www.lendingclub.com/info/download-data.action

data\LCDataDictionary.xlsx explains the definitions used in the LoanStat3a file. The LoanStat3a file was converted using encodings into lendingclubtraining.csv and lendingclubtest.csv
